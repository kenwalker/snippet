Snippet : Code Snippet Editor Plugin for Orion

This is a plugin for Eclipse Orion. The plugin adds a command to the Orion Editor for creating an HTML fragment to show your code snippet in an HTML page. You just highlight the code you want and press the Snippet button.

Install into OrionHub or another Orion instance

Here's an example Snippet generated from the Snippet plugin

    var serviceProperties = {
        name: 'Snippet',
        tooltip: 'Create a JavaScript Orion Editor Snippet for a Web Page',
        key: ['S', true, true] // Ctrl+Shift+S
    };
    provider.registerServiceProvider('orion.edit.command', serviceImpl, serviceProperties);
